# Variáveis
Imagine que você recebeu a tarefa de criar um algoritimo para armazenar  pacientes de um hospital.

Sendo assim crie abaixo as variáveis para armazenar: 

1. **Idade do paciente;**
2. **Nome Completo;**
3. **Altura do paciente;**
4. **Peso;**
